- [ ] Java中的位运算
    表示方法：Java中二进制数据采用补码表示。即正数最高位为0，其余各位代表数值本身，负数通过对该数的绝对值取反，再加1。
    位运算符有：~(非)、>>(位移)、>>>(无符号位移)、&(与)、^(异或)、|(或)
